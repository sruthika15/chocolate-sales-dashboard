# chocolate-sales-dashboard
Chocolate Sales Analysis Dashboard - Power BI project analyzing sales performance with KPIs, trends, and insights across salespeople, products, and countries.
# Project objective
This project analyzes chocolate sales data across multiple dimensions including salespeople, countries, products, and time periods. The dashboard delivers key performance indicators (KPIs) and answers critical business questions to drive data-informed decisions.
## Dataset used
<a href=https://github.com/sruthika15/chocolate-sales-dashboard/blob/main/Chocolate%20Sales%20(2).csv>dataset</a>
## Questions(KPI's)
1. Total Revenue 
2. Revenue Growth Rate (MoM)
3. Average Revenue per Box
4. Revenue per Salesperson
5. Top 5 and bottom 5 performing salespeople
6. Best and worst performing products
7. Most profitable countries
8. Sales trends over time

1. Data Understanding

3. KPI Development
Created 4 core KPIs using DAX:
Total Revenue
Revenue Growth Rate (MoM)
Average Revenue per Box
Revenue per Salesperson

4. Analytical Framework
Structured analysis around 4 key dimensions:
Performance Analysis - Overall trends and rankings
Product Analysis - Product mix and pricing
Geographic Analysis - Country-level insights
Salesperson Analysis - Team productivity metrics

5. Dashboard Creation
Built interactive visualizations (bar charts, line charts, cards)
Implemented Top N filtering for rankings
Added slicers for dynamic filtering by Date, Country, Product
Created KPI cards with growth indicators

6. Insights Generation
Answered 30+ business questions including:
Top 5 performing salespeople by revenue
Product contribution to total sales
Seasonal trends and patterns
Geographic market opportunities
Team performance distribution

7. Deliverables
Interactive Power BI dashboard (.pbix file)
Documented DAX measures
Business insights and recommendations
